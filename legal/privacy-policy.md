# Privacy Policy

**Last updated:** January 1, 2025

## 1. Information We Collect
- **Account data:** Email address, username, hashed password
- **Usage data:** API requests, workflow executions, token usage
- **Technical data:** IP address, browser type, device information
- **Content data:** Prompts, agent configurations, uploaded documents

## 2. How We Use Your Information
- Provide and improve the NexusAI platform
- Send service notifications and security alerts
- Analyze usage to improve performance
- Comply with legal obligations

## 3. Data Retention
- Active accounts: Data retained while account is active
- Deleted accounts: Data deleted within 30 days
- Audit logs: Retained for 7 years (regulatory requirement)
- Backup data: Retained for 30 days

## 4. Your Rights (GDPR)
- **Access:** Export all your data via Settings → Export Data
- **Deletion:** Delete your account and all data via Settings → Delete Account
- **Portability:** Download your data in JSON format
- **Correction:** Update your profile at any time
- **Objection:** Opt out of analytics in Settings → Privacy

## 5. Data Security
- All data encrypted in transit (TLS 1.3)
- Sensitive data encrypted at rest (AES-256)
- API keys hashed with bcrypt before storage
- Regular security audits and penetration testing
